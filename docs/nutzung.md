# Nutzung im Chat

Du kannst HAcoBot über zwei Wege erreichen:

- Über das **Assist-Symbol** (Lupe / Sprechblase) oben rechts in Home Assistant
- Über die **HAcoBot Dashboard-Karte**

## Beispiel-Befehle

### System & Updates

- „Gibt es Updates?“
- „Installiere alle Updates.“
- „Starte das System neu.“

### Diagnose

- „Wie ist die Lage?“ (Briefing)
- „Check mal die Logs auf Fehler.“
- „Ist das System gesund?“

### Dashboards

- „Erstelle eine Karte für mein Wohnzimmer mit allen Lichtern.“
- „Erstelle eine sun-position-card.“

### Organisation

- „Schreibe Milch auf die Einkaufsliste.“
- „Lösche den Zahnarzt-Termin morgen.“
