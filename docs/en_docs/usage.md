# Use in Chat

You can reach HAcoBot in two ways:

- Via the **Assist icon** (magnifying glass / speech bubble) at the top right in Home Assistant
- Via the **HAcoBot Dashboard Card**

## Example Commands

### System & Updates

- "Are there any updates?"
- "Install all updates."
- "Restart the system."

### Diagnostics

- "What's the status?" (Briefing)
- "Check the logs for errors."
- "Is the system healthy?"

### Dashboards

- "Create a card for my living room with all lights."
- "Create a sun-position-card."

### Organization

- "Add milk to the shopping list."
- "Delete the dentist appointment tomorrow."
