# Installation

Aktuell wird HAcoBot manuell installiert.  
(HACS-Support ist in Planung.)

## Schritt 1: Dateien kopieren

1. Lade das Repository herunter.
2. Kopiere den Ordner `custom_components/hacobot`  
   in dein Home-Assistant-Verzeichnis unter: `/config/custom_components/`

3. Starte Home Assistant neu.

## Schritt 2: Browser-Cache leeren

Da HAcoBot eine eigene Dashboard-Karte mitbringt, **musst du nach dem Neustart zwingend den Browser-Cache leeren**, damit die JavaScript-Dateien korrekt geladen werden.

- Tastenkombination: **STRG + F5**

## Schritt 3: Integration hinzufügen

1. Gehe zu **Einstellungen → Geräte & Dienste**
2. Klicke auf **Integration hinzufügen**
3. Suche nach **HAcoBot**
4. Folge dem Konfigurations-Assistenten

> **Hinweis:**  
> Wenn die Integration nicht in der Liste erscheint, leere den Browser-Cache erneut und lade die Seite neu.
